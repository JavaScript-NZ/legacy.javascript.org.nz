Hi. And Welcome.

The JavaScript Society of New Zealand is a non profit organisation set up to grow the JavaScript community within New Zealand. This includes running conferences and meet ups, and also providing help to those learning and using our favourite little programming language.

We're active on the JavaScript NZ Slack channel and welcome you to come join the conversation. If you would like to support the society then you can become a member for a small yearly fee.

Upcoming events

nz.js(con); New Zealand's first dedicated national JavaScript conference will be held on March 9th & 10th, 2017 at Shed 6 in Wellington. It has an open call for papers, a low cost of entry, and will have a broad variety of JavaScript related topics. If you want to know more you can find all the info you need on the nz.js(con); website
